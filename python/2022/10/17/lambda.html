(예비학습) 함수 g(x)g(x)가 정의되어 있을때 $\frac{d}{dx}g(x)$의 값을 계산해보자(도함수)

(목표) 도함수를 구해주는 derivate 함수를 정의하자. 이 함수는 임의의 함수 g를 입력으로 받으면, g의 도함수(gg)가 리턴되는 기능을 가진다.

def derivate(g):
    return lambda x: (g(x+0.001)-g(x))/0.001
